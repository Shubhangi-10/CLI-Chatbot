#CLI Chatbot

A tiny command-line chatbot powered by Google's Gemini API. Runs locally, talks back, remembers the conversation while it's open.

##Getting it running
run the chatbot.py file in the terminal
python chatbot.py

Type away. Type `exit` or `quit` whenever you're done.

##A few things worth knowing
- The chat remembers earlier messages in the same session — close the script and that memory's gone.
